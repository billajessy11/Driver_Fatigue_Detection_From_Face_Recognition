# Driver_Fatigue_Detection_From_Face_Recognition
Developed an AI-based Driver Fatigue Detection System to help prevent road accidents by monitoring driver alertness in real time. The project uses face recognition and behavioral analysis techniques to detect signs of drowsiness such as eye closure, yawning, and head movement. Implemented a hybrid deep learning model using CNN and LSTM, where CNN extracts facial features from video frames and LSTM analyzes temporal patterns over time for accurate fatigue prediction. The system provides instant alerts when fatigue is detected, improving driver safety and reducing accident risks. Built using Python, OpenCV, TensorFlow/Keras, and deep learning techniques.
